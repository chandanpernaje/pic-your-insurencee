# Pic Your Insurance

## Overview

Pic Your Insurance is a modern insurance portal designed for customers to explore, compare, and inquire about various insurance policies online. The platform is built for **Nandan Kumar Pernaje (IRDAI Certified POSP)** and provides a simple way for users to connect with an insurance advisor for policy quotes, renewals, and claim assistance.

## Features

* Vehicle Insurance

  * Car Insurance
  * Bike Insurance
  * Auto Insurance
  * Bus Insurance

* Health Insurance

* WhatsApp Integration for instant support

* Mobile Responsive Design

* English and Kannada Language Support

* Fast and User-Friendly Interface

## Technology Stack

### Frontend

* React
* TypeScript
* Vite
* Tailwind CSS

### Routing

* TanStack Router

### Backend & Database

* Supabase

### Deployment

* Cloudflare Pages

## Project Goals

* Simplify insurance policy inquiries.
* Provide quick access to insurance quotes.
* Improve customer communication through WhatsApp.
* Offer a professional digital presence for insurance services.

## Installation

```bash
git clone https://github.com/chandanpernaje/pic-your-insurencee.git

cd pic-your-insurencee

npm install

npm run dev
```

## Build for Production

```bash
npm run build
```

## Deployment

This project is configured for deployment on Cloudflare Pages.

```bash
npm run build
```

Upload the generated `dist` folder to Cloudflare Pages or connect the GitHub repository directly.
